+ chorme,safari:webkit内核
+ firefox:Gecko内核
+ IE: Trident内核
+ 360,搜狗等国内浏览器:trident+webkit

+ 浏览器内核由很多模块组成\
    - js引擎模块:负责js程序的编译与运行
    - html,css文档解析模块:负责页面文本的解析
    - DOM/CSS模块:负责dom/css在内存中的相关处理. 生成了dom对象
    - 布局和渲染模块:负责页面的布局和效果的绘制(内存中的对象)  这些都是主线程
    - ....  
    - 定时器模块,负责定时器的管理
    - 事件响应模块: 负责事件的管理
    - 网络请求模块:负责ajax请求
    - 分线程

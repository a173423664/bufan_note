>闭包的缺点
+ 函数执行完后,函数内部的局部变量没有释放,占用内存时间变长.
+ 容易造成内存泄漏
>解决
+ 能不用闭包就不用
+ 及时释放

>内存溢出(错误)
+ 一种程序运行出现的错误
+ 当程序运行的内存超过了剩余的内存,就会抛出内存溢出的错误

>内存泄露 
+ 占用的内存没有及时释放
+ 内存泄露积累多了就容易导致内存溢出
+ 常见的内存泄露
    - 意外的全局变量
    - 没有及时清理的计时器或回调函数.
    - 闭包
